# Community Deployment Files

This directory contains community-proposed installers, deployment helpers, experiments, and platform alternatives.

These files are not automatically tested, supported, or distributed by Feedline Labs. Nothing in this directory should be included in an official AirPulse build unless it has been reviewed, tested, and deliberately promoted into `/FeedlineLabs`.

Create a separate descriptive subfolder for each contribution:

```text
Community/<Platform>/<Contribution-Name>/
```

Each contribution must include its own `README.md` describing compatibility, prerequisites, installation, testing, rollback, and known limitations.
